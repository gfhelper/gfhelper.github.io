Code / Issue Contribute
=======================

由于精力有限，原作者无法继续长时间开发后续功能。但是会做代码的维护和
Code Review 。

本项目欢迎贡献代码和 issue 。

代码贡献者请在测试无误后发起 PR 。

-  请贡献风格良好的代码：命名规范、结构清晰且代码已经 ``gofmt``
   格式化过。
-  请考虑较为 General 的 idea
   ，若您的修改或者新功能只对很少数的人有用，并会打扰到大多数人的使用，PR可能不会通过。

Issue
提出者请清晰地描述您的问题，并贴出详细的输入输出（请注意您的账号隐私，关键部分请自行马赛克）。

本项目\ **不接受捐助**\ ，并\ **禁止将本项目代码或本项目的衍生项目用于盈利目的**\ 。本项目采用
AGPL
开源协议，如果您基于本项目做了后续开发，那么您必须将其开源，并保留代码原始版权声明。
